 О проекте
Проект представляет собой многофункциональный веб-ресурс с:
- Информационной страницей — биография Эми, хронология событий, подозреваемые
- Форумом — обсуждение теорий с системой тем и сообщений
- Системой авторизации — роли пользователей (админ, модератор, пользователь)
Функционал
 Модуль | Возможности 
Главная страница - Биография, хронология, подозреваемые, форма отправки теорий 
Форум - Создание тем, отправка сообщений, поиск по темам 
Авторизация - Регистрация, вход, система ролей, выход 
Модерация Одобрение/отклонение теорий, удаление сообщений, блокировка пользователей 
Админ-панель Просмотр всех теорий, статистика, управление пользователями
 Технологии
 Категория | Технологии 
Frontend - HTML5, CSS3 (Flexbox, Grid), JavaScript
Backend - PHP 8.0+, PDO для работы с БД 
База данных - MySQL 8.0 
Сервер- Open Server (Apache + MySQL)
Дополнительно - Сессии PHP, localStorage, AJAX (fetch API)

Установка и запуск
 Требования
- Open Server (или аналогичный локальный сервер с PHP + MySQL)
- Браузер с поддержкой современных стандартов (Chrome, Firefox, Edge)

Шаги установки
1. Клонируйте репозиторий
     bash
   git clone https://github.com/driklina/amybradley-case-.git
   cd amybradley-case-

2.Настройте виртуальный хост в Open Server
  Домен: amybradley.local
  Папка: путь к проекту
3.Добавьте в hosts файл: 127.0.0.1 amybradley.local
  Импортируйте базу данных
  Откройте http://127.0.0.1/phpmyadmin/
  Создайте базу данных amybradley (кодировка utf8mb4_unicode_ci)
  Импортируйте файл database.sql
4.Запустите проект
  Откройте в браузере: http://amybradley.local/
